## 1. 목적

- LLM 여러 개와 동시에 대화하며 **브레인스토밍**을 수행한다.
- 머릿속에 정리되지 않은 **아이템/프로젝트를 정리**하는 것이 목표다.
- **둘 이상의 LLM**이 참여한다.

---

## 2. 기능 요구사항

| FR-ID | No | 분류 | 요구사항 | 비고 |

## 3. 산출물

| 산출물 | 설명 | 저장 |
|--------|------|------|
| `chatgpt.md`, `claude.md`, `gemini.md` … | 참여 LLM별 대화 흐름 (Request / Response / Request(+타 LLM 응답)) | 저장 |
| `summary.md` | 최종 정리본 (아이템/프로젝트 종합) | 저장 |
| 비밀채팅 | 특정 LLM과 1:1 별도 대화 | **저장 안 함 (휘발성)** |

---

## 4. 참고 — 토큰 환산

- 1000토큰 ≈ 한글 700~1200자 (LLM별 토크나이저 상이, 브라우저 방식은 글자수 근사 가능)
- 100토큰 ≈ 한글 70~120자

---

## 5. 미해결 / 후속 검토

- **크롬 확장 기술검토**
  - content script 주입
  - 각 LLM 사이트 DOM 대응
  - 토큰 카운트 근사
- 설계 / 유사코드는 **추후 단계**에서 진행
