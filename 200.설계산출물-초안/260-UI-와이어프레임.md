## 레이아웃 규칙 [v2]

| 요소 | 위치 | 설명 |
|------|------|------|
| LLM 풀 패널 | 좌측 10% | LLM pool 체크박스 (6종), 전체선택 |
| 설정 바 | 우측 상단 | 라운드/토큰 select, 세션 종료 |
| LLM 슬롯 그리드 | 우측 중앙 90% | 3열×2행 (최대 6개), 각 슬롯: 헤더(LLM명+swap)+응답+공유/지우기/기록/🔒 |
| 프롬프트 입력 | 우측 최하단 | input + 전송 버튼 |
| 비밀채팅 | 오버레이 | 메인 화면 위에 표시 |
| 저장 화면 | 다이얼로그 | 세션 종료 시 |
