# 내가 원하는 것 — AI별 정리 요약

---

## 공통 핵심 (4개 AI 모두 동일)

> 브라우저 로그인 세션 기반으로 접근 가능한 LLM을 탐지·선택하고, 동기식 3라운드 구조로 협업시킨 뒤, 사용자가 최종 판단하는 멀티 LLM 플랫폼

---

## AI별 요약 방식 차이

| AI | 요약 스타일 |
|---|---|
| **Claude** | 가장 간결. 규칙과 흐름을 bullet로 정리 |
| **Gemini** | 제품 철학 중심. HITL + SI 파이프라인 강조 |
| **ChatGPT** | 목표 정의 중심. 플랫폼 개념으로 서술 |
| **DeepSeek** | 구현 단위 분해 중심. 코드/기술 스택 포함 |

---

## 핵심 규칙 (공통 확정)

| 항목 | 내용 |
|---|---|
| LLM 접속 | 브라우저 로그인 세션 (API Key 불필요) |
| LLM 선택 | 탐지된 목록에서 일부 또는 전체 선택 |
| Round 1 | 독립(Blind) 응답 |
| Round 2 | 상호 검토·비판 |
| Round 3 | 최종 제안 |
| 동기화 | 모든 응답 완료 후 다음 라운드 진행 |
| 최종 판단 | 사용자 직접 (AI 자동 합의 없음) |
