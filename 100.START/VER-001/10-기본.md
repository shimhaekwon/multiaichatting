# 다자 AI 브레인스토밍 — 기본 개요

## 한 줄 요약
크롬 확장 프로그램. 최대 10종 LLM에 동시 질문하고, 라운드 단위로 응답을 서로 공유하며 브레인스토밍하는 도구. ChatHub 방식.

## 핵심 동작
- **로그인 세션 기반** — API 키 불필요, 브라우저 로그인 활용
- **대상 LLM 10종**: chatgpt, gemini, claude, deepseek, mistral, grok, kimi, perplexity, pi, qwen
- **Top 8까지만 노출**, **2개 이상 선택 필수**
- **라운드 반복** (시작 시 3~9 지정)
  - 각 LLM 1회 응답 (1000토큰 제한)
  - 타 LLM 응답을 다음 라운드에 전달
  - 응답 순서 제한 없음
- **STOP 버튼** — 사용자 수동 클릭 (자동 stop 없음)
- **비밀채팅** — 특정 LLM과 1:1 별도 대화 (저장 안 함)

## 저장/산출물
- 자동 저장 없음 (사용자 요청 시 MD 저장)
- LLM별 MD 파일 + 최종 정리본(summary.md)

## 기술 제약
Chrome 전용 / Manifest V3 / 단일 사용자 / 로컬 전용 / 모바일 미지원

## 미해결
content script 주입, 각 LLM DOM 대응, 토큰 카운트 근사
