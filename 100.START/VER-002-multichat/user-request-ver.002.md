# 多者 AI Cowork 브레인스토밍 모드 — 요구사항 정의 (ver.002)

> 출처: `100.START/user_request-ver.002.txt` (ver.001 = `user_request.txt` 수정본)
> 작성일: 2026-05-29

## 목적
- LLM 여러 개와 동시에 대화하며 **brainstorming** 수행
- 머릿속에 정리되지 않은 **아이템/프로젝트를 정리**
- **하나 이상**의 LLM이 참여 *(ver.001: "둘 이상" → ver.002: "하나 이상")*

## 기능 요구사항

| No | 분류 | 요구사항 |
|----|------|----------|
| 1 | 응답 제약 | LLM은 라운드당 응답 **1회만** 가능 |
| 2 | 토큰 제약 | 응답 **1000토큰 초과 금지** (라운드마다 request에 자동 추가). **토큰 combobox = {100, 500, 1000, 3000, 5000, 10000}** |
| 3 | 라운드 간 전달 | User request와 함께 LLM 응답을 **타 LLM에게 전달**(라운드 단위, **자기 응답 제외**). **포맷 = YAML**, **1라운드는 response 비어있음** |
| 4 | 라운드 제어 | 라운드 수 제한: 시작 시 지정 **combobox = {3, 5, 7, 9}**, 중간 STOP 가능 |
| 5 | 접속 방식 | **chathub 방식** — 크롬 브라우저에 **로그인한 계정**으로 접근 가능한 LLM과 대화 (**API키 없음**) *(ver.001: "google 계정" → ver.002: "로그인한 계정")* |
| 6 | LLM 노출 | LLM 목록은 **우선 6개까지만** 노출 *(ver.001: top 8 → ver.002: 6개)* |
| 7 | 응답 순서 | LLM 응답 **순서 제한 없음** |
| 8 | 구현 형태 | **크롬 확장 프로그램**으로 개발 |
| 9 | 비밀채팅 | **비밀채팅** — 특정 LLM과 1:1 별도 대화 (전체 맥락과 분리, **1000토큰 제한**, **저장 안 함 / 휘발성**) |
| 10 | 대상 LLM | **우선 6종**: chatgpt, gemini, claude, deepseek, perplexity, grok / **차기**: (mistral, kimi, pi, qwen) — 개발 편의를 위해 6종 먼저 |
| 11 | STOP | **STOP 기능** — 응답이 너무 길거나 느릴 때 사용자가 STOP 버튼 **수동 클릭** (자동 stop 없음) |
| 12 | 산출물 | **LLM별 md 파일**(`LLM명.md`) + `summary.md`. 구조 = Request / Response / Request(+다른 LLM 응답) 흐름 |
| 13 | 산출물 | **최종 정리본 md** 생성 (브레인스토밍 결과 종합) |
| 14 | 추가/수정 | **14-1.** 1라운드부터 각 LLM에게 (항목3) 데이터를 전달 — **1라운드에는 LLM만 있고 값은 비어있음**<br>**14-2.** 항목3을 **YAML 포맷**으로 지정 (LLM에게 전달하는 데이터만 **context**) |

## 항목 3 전달 데이터 예시 (YAML)

```yaml
# gemini 에게 (gemini 자기 응답 제외)
chatgpt: 그건 아닌거 같은데
claude: 그럴 수도 있어
```
```yaml
# claude 에게
chatgpt: 그건 아닌거 같은데
gemini: 그럴 수도 있어
```
```yaml
# chatgpt 에게
gemini: 고민스럽네
claude: 그럴 수도 있어
```
> 1라운드에는 위 구조에서 값이 비어있음(LLM 키만 존재). 2라운드부터 이전 라운드 타 LLM 응답으로 채워짐.

## 산출물

| 산출물 | 설명 | 저장 |
|--------|------|------|
| `chatgpt.md`, `gemini.md`, `claude.md` … | 참여 LLM별 대화 흐름 (Request / Response / Request(+타 LLM 응답)) | 저장 (요청 시) |
| `summary.md` | 최종 정리본 (아이템/프로젝트 종합) | 저장 (요청 시) |
| 비밀채팅 | 특정 LLM과 1:1 별도 대화 | **저장 안 함 (휘발성)** |

## 참고 — 토큰 환산
- 1000토큰 ≈ 한글 700~1200자 (LLM별 토크나이저 상이, 브라우저 방식은 글자수 근사 가능)
- 100토큰 ≈ 한글 70~120자

## 변경 이력 (ver.001 → ver.002)

| # | 항목 | ver.001 | ver.002 | 영향 |
|---|------|---------|---------|------|

| 10 | 대상 LLM | 10종 | **6종 우선** + 4종 차기 | 스코프 확정 |
