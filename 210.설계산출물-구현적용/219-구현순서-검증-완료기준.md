## 2. 검증 시나리오 (E2E)

```
1) LLM풀에서 ChatGPT·Claude·Gemini 선택 → 사이드바 3 체크
2) 설정바: 라운드=3, 토큰=1000 → "전송"에 prompt "시장 조사 아이디어를 내줘"
3) status WAITING → 3 셀 동시 스트리밍 → ALL_COMPLETE → ROUND_COMPLETE
4) "다음 라운드" → 각 봇 prompt에 "타 2종 응답 YAML"(자기 제외) 주입 확인
5) 3라운드 도달 → DONE → 저장 다이얼로그 → 4개 md 저장 → 내용 검증
   - chatgpt.md: Round1 Req/Resp, Round2 Req(+타LLM)/Resp 흐름
   - summary.md: 라운드순 전체 합본(각 라운드 Request + 참여 LLM 전원 응답 원문, 214 [G7])
6) (별도) WAITING 중 STOP → **STOP 전 자연완료만 status='completed'**, 나머지 'stopped'(부분/중단 응답 폐기 — VER-001 3.3 정렬, [M2 critic]) → 부분 라운드 history 기록 → 계속하기/종료
7) (별도) 🔒 비밀채팅: Claude와 1:1 → 닫기 후 저장 안 됨 확인
```

## 3. 정적 검증 (코드생성원칙 Step 4)

- **4.1 Code**: `tsc --noEmit` 0, ESLint 신규 위반 0, 한글 하드코딩 0(i18n), TODO/FIXME 잔존 0.
- **4.2 Side-Effect**: 제거한 심볼(premium/plausible/sentry/삭제 봇 + **proxyFetch·chatgpt-inpage-proxy·chatgpt-webapp·claude-web·perplexity-web 등 내부 API 잔재** + **[M-1] `~services/prompts`·`PromptLibrary`·`PromptCombobox`·`use-premium`·`use-purchase-info`·`server-api`(Campaign)·`storage/{open-times,token-usage}`·`release-notes`**) `grep` → 잔존 참조 0. **특히 `ChatMessageInput`(prompts/PromptLibrary)·`abstract-bot`(sentry)·`History/Dialog`(use-premium)·`state/index.ts`(Campaign/FeatureId)는 "재사용"이라도 import 정리 필수.** `createBotInstance`/`CHATBOTS`/`BotId` 호출처 정합.
- **4.3 Existed**: `AbstractBot` 이벤트 계약(UPDATE_ANSWER/DONE/ERROR) 준수, `secretChatAtom`이 `atomWithStorage` **미사용**(휘발)으로 **비밀채팅 저장 안 함** 확인, 6사이트 host permission + `dom-adapter.ts` 주입 경로 정합.
