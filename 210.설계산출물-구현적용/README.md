## 핵심 위험 (반드시 인지)

> **[2026-05-29 전략 전환]** Phase 0 실증 결과 ChatHub 1.45.7의 내부 API 봇(ChatGPT/Claude/Perplexity)이 현재 사이트와 안 맞아 **전부 실패** 확인(공개 OSS는 ~2025-03 동결, 최신은 비공개 4.1.1). → **하이브리드 폐기, 6종 전부 DOM 어댑터로 통일.** 내부 API 재사용 가정 무효.

1. **[selector 노후]** 6종 모두 DOM → 사이트 개편 시 selector 갱신 필요. `adapter-config.json`(version·**단일 출처**, 봇 클래스 미보유) 분리로 **로직 재작성 없이** config 수정으로 대응(217 §3.2/§3.3 [M-2]).
2. **[탭 의존]** DOM 봇은 각 사이트 탭이 열려 있고 로그인돼야 함 → 미로그인 시 "로그인 필요" 표시.
3. **[ToS 회색지대]** 무키 웹 자동화는 각 사 ToS상 회색지대 — 단일 사용자 내부용 전제.
4. **[원안 부분수정]** 원안 SW 라운드엔진 → 앱 컨텍스트 엔진(213). 사용자 인지 완료.
