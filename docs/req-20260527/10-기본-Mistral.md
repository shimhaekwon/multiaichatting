### **10-기본-고객요구.md**
**목적**: 사용자 및 비개발자 이해를 위한 간결한 요약
**내용**:
- **시스템 이름**: ChatHub (크롬 확장 플러그인으로 실행)
- **기능 요약**:
  - 3종 LLM(Claude, ChatGPT, Gemini)를 로그인 세션 기반으로 탐색 (API 키 불필요).
  - 사용자가 **2개 이상 LLM을 선택** 후, **3~9회 라운드**로 구조화된 채팅 진행.
    - R1: 각 LLM 독립적으로 답변 (Blind).
    - R2: 다른 LLM의 R1 답변을 기반으로 의견 제시.
    - R3: 최종 답변.
  - **자동 저장** (IndexedDB) + 사용자 설정 가능 (타임아웃, 재시도 횟수).
  - **MD 저장** 기능과 이력 조회 기능 제공.
- **제약**:
  - Chrome 전용 (Manifest V3), 단일 사용자, 로컬 저장만 지원.
