# 멀티-LLM 라운드 토론 시스템 — 기본 개요

## 한 줄 요약
크롬 확장 플러그인. 여러 LLM(Claude/ChatGPT/Gemini) 동시에 질문 보내고, 라운드 돌려서 서로 답 비교/검토 시키는 도구. ChatHub 개선판.

## 핵심 컨셉
- **로그인 세션 기반** — API 키 안 씀, 브라우저 로그인 그대로 활용
- **3개 LLM 자동 탐색** — Claude, ChatGPT, Gemini
- **2개 이상 선택 필수** — 단독 실행 불가
- **3라운드 기본 / 최대 9라운드**
  - R1: Blind 독립 응답
  - R2: 타 LLM 응답 검토 후 의견
  - R3: 최종 답변
- **모든 LLM 응답 완료 후 다음 라운드 진행** (동기화)

## 운영 규칙
- 타임아웃 기본 5분 (사용자 조정 가능)
- 라운드 사이에만 LLM 추가/제외, 재질의 가능
- IndexedDB 자동 저장 + MD 파일 내보내기
- 출력 형식 강제 (구조화, 1500 토큰 제한)

## 기술 제약
- 크롬 전용 / Manifest V3
- 단일 사용자 / 로컬 전용 / 서버 없음
- 모바일 미지원
