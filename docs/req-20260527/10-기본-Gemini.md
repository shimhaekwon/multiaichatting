# [개요] LLM 다중 라운드 토론 시스템 (v2.0)

본 문서는 사용자가 제안한 크롬 확장 프로그램 기반의 'LLM 다중 라운드 토론 시스템' 개발을 위한 개요 서명서입니다. 개발 원칙에 의거하여 추정치 없이 요구사항을 명확하게 요약했습니다.

## 1. 시스템 핵심 목적
사용자가 크롬 브라우저 로그인 세션을 기반으로 3대 LLM(Claude, ChatGPT, Gemini)을 연동하여, 다중 라운드(기본 3회, 최대 9회) 방식을 통해 상호 교차 검증된 최종 답변을 도출하는 로컬 전용 브라우저 확장 프로그램입니다.

## 2. 주요 기능 요약
* **LLM 세션 연동:** API 키 없이 크롬 브라우저의 기존 로그인 세션을 자동으로 탐색 및 활용합니다. (ChatHub 오픈소스 기반 개선)
* **다중 라운드 토론 규칙:** * **R1 (1라운드):** 타 LLM의 답변을 모른 채 독립적으로 응답 (Blind)
  * **R2 (2라운드):** 타 LLM의 1라운드 답변 전체를 검토 후 상호 의견 제시
  * **R3 (3라운드) ~ 최대 9라운드:** 최종 답변 도출 및 사용자 제어에 따른 회차 증감/연장/조기종료
* **사용자 제어 및 개입:** 라운드 간 대기 시간에 LLM 변경, 개별 1:1 재질의, 다음 회차 질문 사전 입력이 가능합니다. 단, LLM 응답 중에는 개입이 불가합니다.
* **로컬 데이터 관리:** 서버 없이 사용자의 브라우저 내 IndexedDB에 라운드별로 자동 저장되며, 전체 이력 조회 및 Markdown(종합/LLM별 개별) 파일 내보내기를 지원합니다.

## 3. 기술 제약 및 환경
* **플랫폼:** Chrome 브라우저 전용 확장 프로그램 (Manifest V3 규격)
* **지원 OS:** Windows, macOS, Linux (모바일 미지원)
* **아키텍처:** 단일 사용자용, 로컬 전용 아키텍처 (서버 저장 없음, API 키 미사용, 암호화 미적용)
