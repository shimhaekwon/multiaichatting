## 1. 개요

### 1.1 목적
복수 LLM의 독립 응답과 상호 검토를 통해 단일 LLM 사용 대비 답변 품질 향상.

### 1.2 범위
- 포함: 크롬 확장프로그램, 웹UI, 3라운드 협업 엔진, 로컬 저장
- 제외: 모바일 지원, 멀티 사용자, 서버 저장, API 키 기반 호출

### 1.3 용어
- **LLM 세션:** 사용자가 크롬에서 로그인한 LLM 서비스
- **라운드:** 모든 선택 LLM이 동기화되어 진행하는 응답 단계
- **블라인드 응답:** 타 LLM 응답을 보지 않고 작성한 독립 응답
- **재질의:** 특정 LLM에 대한 1:1 추가 질문

## 2. 시스템 구성

### 2.1 구성 요소
- 크롬 확장프로그램: LLM 탭 제어, 메시지 송수신
- 웹UI: 사용자 인터페이스, 협업 흐름 제어
- 로컬 저장소: IndexedDB

### 2.2 동작 환경
- 브라우저: Chrome 최신 안정 버전
- OS: Windows / macOS / Linux
- 모바일: 미지원

### 2.3 지원 LLM
- Claude
- ChatGPT
- Gemini

## 3. 기능 요구사항

### 3.1 LLM 접속 및 탐색
- **FR-001:** 크롬 로그인 세션 기반 LLM 접근 (API 키 미사용)
- **FR-002:** 지원 LLM 3종에 대한 로그인 상태 자동 탐색
- **FR-003:** 탐색 결과를 사용자에게 목록 표시 (로그인 여부 포함)

### 3.2 LLM 선택
- **FR-010:** 탐색된 LLM 중 일부 또는 전체 선택
- **FR-011:** 최소 2개 이상 선택 필수
- **FR-012:** 선택 상태를 세션 간 유지 (IndexedDB 저장)

### 3.3 라운드 진행
- **FR-020:** 최대 3라운드 진행
- **FR-021:** 라운드 동기화 - 모든 선택 LLM 응답 완료 후 다음 라운드 진행
- **FR-022:** R1 - 블라인드 독립 응답 (타 LLM 응답 미공유)
- **FR-023:** R2 - 타 LLM의 R1 응답 전체 텍스트 검토 후 의견 제시
- **FR-024:** R3 - 최종 답변 작성
- **FR-025:** 사용자 판단에 의한 라운드 조기 종료 가능

### 3.4 사용자 개입
- **FR-030:** 각 라운드 시작 전/후 사용자 개입 가능
- **FR-031:** 특정 LLM 1:1 재질의 기능
- **FR-032:** 재질의 결과는 다음 라운드 컨텍스트에 포함
- **FR-033:** 최종 판단 권한은 사용자

### 3.5 뷰 및 산출
- **FR-040:** LLM별 전체 대화 팝업 뷰
- **FR-041:** MD 파일 다운로드 저장
- **FR-042:** MD 포맷 - 라운드별 헤더 + LLM별 응답 블록 구조

### 3.6 데이터 관리
- **FR-050:** 대화 이력 IndexedDB 저장
- **FR-051:** 라운드 완료 시마다 자동 저장
- **FR-052:** 이력 조회 기능 제공 (최소 기능: 목록 + 열람)
