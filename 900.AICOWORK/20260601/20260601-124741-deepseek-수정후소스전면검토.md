```typescript
chrome.runtime.onConnect.addListener((port) => {
  let currentSelectors: AdapterSelectors | undefined;
  let stopObserve: (() => void) | null = null; // ← hoist

  port.onMessage.addListener(async (msg: any) => {
    if (msg.action === "SEND_PROMPT") {
      // ... (변경 없음) ...
      stopObserve = observeResponse(s, (text, done) => {
        // ← const→let, 외부 변수 할당
        port.postMessage(
          done
            ? { action: "DONE" }
            : { action: "RESPONSE_CHUNK", content: text },
        );
      });
      port.onDisconnect.addListener(() => {
        stopObserve?.(); // ← null 안전 + cleanup
        stopObserve = null;
      });
    } else if (msg.action === "STOP") {
      if (currentSelectors) findStop(currentSelectors)?.click();
      stopObserve?.(); // ← NEW: 타이머 즉시 중단
      stopObserve = null;
      port.postMessage({ action: "DONE" }); // ← NEW: 스트림 종료 통보
    }
  });
});
```

| 경로                        | 동작                                                                                    | 이상 유무  |
| --------------------------- | --------------------------------------------------------------------------------------- | ---------- |
| 자연 done (observeResponse) | onUpdate(done=true) → DONE 전송 → port.onDisconnect → stopObserve?() (이미 clear, 무해) | ✅ 이상 無 |
| STOP 요청                   | stopObserve?()로 timer 중단 + DONE 전송                                                 | ✅ 이상 無 |

### 1.3 `background/index.ts` ✅ M1 + 신규 cleanup

**M1 — connectToCSWithRetry 직렬화**:

```typescript
// 전: return new Promise(...) → 루프 첫 회 탈출, 재시도 실효 無
// 후: await + null 체크 → 실패 시 루프 계속, 최대 3회 재시도
for (let i = 0; i < retries; i++) {
  try {
    const port = chrome.tabs.connect(tabId, { name });
    const result = await new Promise<chrome.runtime.Port | null>((resolve) => {
      port.onDisconnect.addListener(() => resolve(null));
      setTimeout(() => resolve(port), 100);
    });
    if (result !== null) return result; // ← 성공 시 탈출
  } catch {
    /* tabs.connect failed */
  }
  if (i < retries - 1) await new Promise((r) => setTimeout(r, delay));
}
```

**타이밍**: 100ms 판정 + 300ms delay × 최대 3회 ≈ 최대 1.2초 대기. best-effort.

**신규 — cleanup appPort ERROR 통보**:

```typescript
const cleanup = () => {
  appPort.onMessage.removeListener(stopListener);
  try {
    csPort.disconnect();
  } catch {
    /* already gone */
  }
  try {
    appPort.postMessage({
      action: "ERROR",
      code: "ERR_002",
      message: "LLM tab connection lost",
    });
  } catch {
    /* appPort may be gone */
  }
};
```

**영향 경로**: CS 탭 종료 → csPort.onDisconnect → cleanup → `appPort.postMessage(ERROR)` → DomAdapterBot.onMessage → `params.onEvent(ERROR)` → `controller.error(error)` → 스트림 종료. C1과 독립적인 hang 경로 차단.
